# 🤖 AI Automation Portfolio: CV Screening Workflow

This project is dedicated to automating the CV screening process using **n8n** and artificial intelligence.

## 📝 Project Description
This workflow receives PDF files of applicants for a particular job, then the AI analyzes the experiences, matches them with the job requirements, and provides a final evaluation.

## 🛠 Tools Used
* **n8n**: for managing the workflow.
* **OpenAI (GPT-4)**: for analyzing CV content.
* **Google Drive / Dropbox**: for receiving files.

## 🚀 How It Works
1. The CV is uploaded.
2. The text is extracted and converted into structured data.
3. The AI compares it with the job description.
4. A notification of the result is sent.
